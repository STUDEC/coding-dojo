    DOMINO enigma coding dojo

    Consider:

    • a 6*6 grid with the two cases (0,0) and (5,5) filled in black
    • 2*1-sized dominos

    Is it possible to cover all the white cases of the grid with the dominos ? (No domino overlaps, no grid overlap)

    The domino project aims to resolve the enigma.

    TDD approach is used.
